# 洛克世界频道 - 需求文档 v1.0

## 产品定位
洛克王国非官方论坛，提供宠物查询和社区交流两大核心功能。

---

## 一、查询插件功能

### 1. 宠物详细信息查询

| 功能点 | 详细描述 | 优先级 |
|--------|---------|:------:|
| 基础信息 | 名称、类别、图片、血脉、资质值等 | P0 |
| 培养推荐 | 推荐性格提升属性、推荐天分提升属性、推荐技能佩戴 | P0 |
| 捕获相关 | 捕获地点、捕获条件、捕获时间 | P0 |
| 进阶相关 | 是否可进阶、可进阶形态、进阶条件 | P0 |
| 异色相关 | 是否有异色形态 | P1 |

### 2. 宠物属性对比工具

| 功能点 | 详细描述 | 优先级 |
|--------|---------|:------:|
| 双宠对比 | 选择两个宠物，横向对比资质值 | P0 |
| 对比展示 | 雷达图/柱状图可视化对比 | P0 |

## 三、技术要求

### 平台要求
- **目标平台**：Android APK
- **显示方式**：悬浮窗横屏显示

### 关键约束
1. 悬浮窗需要横屏适配
2. 需要后台保活机制
3. 消息推送能力（世界频道）

---

## 四、数据需求

### 宠物数据字段
```
- 名称（name）
- 类别/系别（category）
- 图片（image）
- 血脉（bloodline）
- 资质值（stats: HP/攻击/防御/魔攻/魔防/速度）
- 推荐性格（recommended_nature）
- 推荐天分（recommended_talent）
- 推荐技能（recommended_skills）
- 捕获地点（capture_location）
- 捕获条件（capture_condition）
- 捕获时间（capture_time）
- 是否可进阶（can_evolve）
- 进阶形态（evolution_forms）
- 进阶条件（evolution_condition）
- 异色形态（albinism_form）
```
