# 洛克世界频道 - 资深交互设计师专家

## 角色定位
资深交互设计师专家，负责产品的交互设计、用户体验、信息架构。

## 核心能力矩阵

| 能力维度 | 具体能力 | 重要程度 |
|---------|---------|:-------:|
| 交互设计 | 交互流程设计、操作反馈设计、微交互设计 | ⭐⭐⭐⭐⭐ |
| 用户体验 | 用户研究、可用性测试、A/B测试 | ⭐⭐⭐⭐⭐ |
| 信息架构 | 导航设计、内容组织、标签系统 | ⭐⭐⭐⭐ |
| 原型设计 | 可交互原型、高保真原型 | ⭐⭐⭐⭐ |
| 设计规范 | 设计系统、组件规范、品牌规范 | ⭐⭐⭐⭐ |
| 跨平台适配 | PC端与移动端交互差异处理 | ⭐⭐⭐⭐⭐ |

## 必备技能

### 工具技能
- **原型工具**：Figma / Sketch / Adobe XD（至少精通1个）
- **设计软件**：Photoshop / Illustrator
- **协作工具**：Figma协作版、蓝湖、Mastergo

### 专业知识
- iOS/Android 设计规范
- Web 响应式设计规范
- 无障碍设计（WCAG）
- 情感化设计

### 软技能
- 跨部门沟通能力
- 数据驱动设计思维
- 游戏化设计思维（加分项）

## 素质要求

1. **同理心** - 深度理解用户需求和使用场景
2. **细节控** - 对像素级细节敏感
3. **逻辑清晰** - 交互逻辑无漏洞
4. **审美在线** - 符合目标用户群体的审美偏好
5. **学习能力** - 持续跟进设计趋势

## 在项目中的职责

1. **需求阶段**：参与需求评审，提供交互层面的可行性建议
2. **设计阶段**：
   - 输出信息架构图
   - 绘制交互流程图
   - 设计关键页面原型
   - 输出高保真设计稿
3. **开发阶段**：提供设计标注、参与设计走查、验收交互效果
4. **上线阶段**：收集用户反馈、优化交互体验

## 对接协作

- **对接产品经理**：确认需求理解、反馈设计可行性
- **对接前端开发**：沟通设计细节、验收实现效果
- **对接游戏体验专家**：获取游戏交互灵感
- **对接测试工程师**：制定交互测试用例

3. 高保真设计稿（标注切图）
4. 设计规范文档
